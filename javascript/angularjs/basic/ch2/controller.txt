1. Controller
	In Angular, a Controller is defined by a JavaScript constructor function that is used to augment the 
	Angular Scope.
	
	The role of controllers in Angular is to expose data to the view via $scope, and to add functions to $scope
	that contain business logic to enhance view behavior.
